# 欢迎使用ZBEE
**ZBEE** 是一款基于控制台作业的工作程化代码设计工具，通过精心的设计与技术实现，可支持所有基于 **Javascript** 的项目代码管理工作 ， 让 **Javascript** 代码也能够象 Java、C# 一样通过命名空间方式来进行工程化管理

## 安装
```
> yarn add zbee -D

```
## 应用
在 **package.json** 中的 **scripts** 字段增加

```
{
    "scripts":{
        ...
        "zb-init":"zb-init",
        "zb":"zb",
        ...
    }
}
```
